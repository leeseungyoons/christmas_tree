# christmas_tree

> 반짝이는 크리스마스 트리입니다.  
> HTML, CSS, JavaScript로 간단하게 구현했습니다.

## 소개

심심해서 만든 미니 작업입니다.  
어두운 배경 위에 반짝이는 별들로 트리 느낌이 나게 표현해봤습니다.

## 파일 구성

- `treehtml.html`
- `tree.css`

## 실행 방법

`treehtml.html` 파일을 브라우저에서 열면 됩니다!
